Travzo Holidays — Complete Website Figma Design Prompt
Brand Foundation
Design a complete travel agency website for Travzo Holidays. The brand is premium, trustworthy, and aspirational — targeting Indian families, couples, and groups. Colour palette: Primary deep navy #1A2A5E, Accent warm gold #C9A227, White #FFFFFF, Off-white #F5F5F5, Dark text #1A1A1A, Muted text #6B7280. Typography: Playfair Display or Cormorant Garamond for headings, Inter or Raleway for body text. Design at 1440px desktop width. All pages must feel part of the same design system — consistent spacing (8px grid), consistent card styles, consistent button styles, and consistent section padding (80px top and bottom).
Button System: Primary button — gold #C9A227 background, navy text, 12px rounded corners, 16px 32px padding. Secondary button — navy #1A2A5E background, white text, same sizing. Ghost button — transparent, navy border, navy text. All buttons have a subtle hover shadow.
Card System: White background, 12px border radius, soft box shadow 0 4px 20px rgba(0,0,0,0.08), 16px internal padding, image at top with rounded top corners, content below.

Final Notes for the Designer
All 10 frames must share the exact same header and footer components — build them as Figma components and reuse. Maintain consistent section spacing of 80px top and bottom throughout. Use 1440px frame width for desktop. Also create a mobile frame at 390px width for the homepage and header only. Use real placeholder text (not lorem ipsum) relevant to travel. All image placeholders should show the subject matter in the placeholder label (e.g. "HERO — dramatic mountain landscape" or "PACKAGE CARD — Bali temple photo"). Use auto layout throughout for all sections and cards so designs are responsive-ready.
